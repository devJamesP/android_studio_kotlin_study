# <이미지 검색기 앱>
  
## 이미지 검색기 앱

-----

### 기능 상세
    * Unsplash Api로부터 사진들을 가져올 수 있다.
    * 검색한 사진을 다운 받을 수 있다.
    * 사진을 배경화면으로 설정할 수 있다.
    * 로딩할 때 Loading Shimmer를 볼 수 있다.

### 활용 기술
    * Retrofit2
    * Coroutine
    * Glide
    * ShimmerLayout
    * WallpaperManager
