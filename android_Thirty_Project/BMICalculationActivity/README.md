  ### BMI Calculation
  Formula = Weight(kg) / Height(m)^2
  weight <= 18.5 : 저체중
  
  18.5 < weight <= 23 : 정상체중
  
  23 < weight <= 25 : 과체중
  
  25 < weight <= 30 : 경도 비만
  
  30 < weight <= 35 : 중정도 비만
  
  35 < weight : 고도 비만
